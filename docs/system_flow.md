System flow plan

# Drivers App
- Login
- Register and onboard options
- Wallets
- Wait for Ride to be assigned
- Start ride
- Broadcast Location for ongoing ride
- Show each student picked up
- End Ride
- Ride history

# Parents App
- Login
- Request Ride
- View Ongoing rides
- Track Ride
- Ride history
- Students or children list

# Admin Panel
- Login
- Dashboard
- View Drivers, Parents
- View Rides
- Assign a ride
- Ride statistics
- Ride history
- Track a ride
